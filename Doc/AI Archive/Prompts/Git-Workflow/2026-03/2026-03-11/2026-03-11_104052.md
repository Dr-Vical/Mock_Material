# Prompt

# Response

완료. 두 가지 변경사항:

1. **A/B/C/D 선택적 트리 업데이트**: 드라이브 버튼(A/B/C/D)이 눌린 상태에서 트리 클릭 시 해당 패널만 업데이트. 해제 상태면 전체 패널 업데이트.

2. **연속→단일 전환**: 연속 수집 중 단일 버튼을 누르면 연속 수집을 중지하고 바로 단일 캡처로 전환. (경고창 없이 즉시 전환)
