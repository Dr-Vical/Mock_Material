# Response

네, Claude Code가 **어디에서 실행되든 항상 읽는 파일**이 있습니다:

### 글로벌 (항상 로드)
- **`~/.claude/CLAUDE.md`** → `C:\Users\bcchoi\.claude\CLAUDE.md`

이 파일은 프로젝트와 무관하게 **모든 대화에서 항상** 로드됩니다.

### 프로젝트별 (해당 프로젝트에서만)
- `{프로젝트}/.claude/CLAUDE.md` 또는 `{프로젝트}/CLAUDE.md`

### 스킬도 마찬가지

스킬을 **글로벌**로 만들려면:
```
~/.claude/skills/{skill-name}/
```
→ `C:\Users\bcchoi\.claude\skills\` 아래에 만들면 어떤 프로젝트에서든 사용 가능합니다.

**프로젝트 전용**이면:
```
{프로젝트}/.claude/skills/{skill-name}/
```
