# Response

현재 구조를 확인했습니다. 지금은 **MaterialDesign PackIcon**으로 아이콘을 표시하고 있고, `IconKind` 문자열 → `PackIconKind` enum 변환으로 동작합니다.

**결론: 가능합니다.** 방법은 두 가지입니다:

### 방법 1: SVG Path를 XAML Path로 변환 (권장)
SVG의 `<path d="...">` 데이터를 WPF `Path.Data`로 직접 사용. 특정 노드 타입일 때 PackIcon 대신 Path를 표시.

### 방법 2: PackIcon + 커스텀 아이콘 병행
노드 타입에 따라 PackIcon 또는 커스텀 SVG Path를 선택적으로 표시.
