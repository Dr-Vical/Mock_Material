# Response

수정 완료.

**1. Oscilloscope 버튼 동작 개선:**
- 첫 클릭: 패널 열림 + 차트 영역 표시 (정지 상태, **Start** 버튼)
- Start 클릭: 그래프 실시간 시작
- 두 번째 Oscilloscope 클릭: 차트 숨김 (+ 타이머 정지)
- Control Panel 열려 있으면 스코프만 닫힘, 양쪽 다 닫히면 패널 전체 숨김

**2. Auto Scale 버튼 추가:**
- 차트 툴바에 Start/Stop 옆에 "Auto Scale" 버튼
- 클릭 시 `Axes.AutoScale()` → 최대값 기준 스케일 조정
