# Prompt

아 즐겨찾기는 잘 만들었어 근데, Favorites 트리 하위에 추가하라는게 아니라 오른쪽 센터 패널 항목들에 추가해 달라고
처음엔 해당 Favorites은 비어 있다가 추가하면 추가되게

---

# Response

변경했습니다. 이제 동작 흐름:

1. **어떤 노드든** 파라미터 목록에서 ★ 클릭 → `_favoriteParameters`에 복사본 저장
2. **다른 노드로 이동** 후 돌아와도 → 이전에 즐겨찾기한 파라미터의 ★ 상태 유지
3. **Favorites 트리 노드 클릭** → 센터 패널에 즐겨찾기된 파라미터만 표시 (처음엔 빈 목록)
4. **★ 해제** → `_favoriteParameters`에서 제거, Favorites 패널에서도 사라짐

Favorites 트리는 하위 노드 없이 리프 노드로 유지됩니다.
