# Prompt

그래프랑 컨트롤 판넬도 멀티로 만들꺼야 우선 좌 하단에 버튼을 a/b/c/d로 분기해서 그래프 a/b/c/d 이렇게 그밑에 chart a/b/c/d 그 밑에 parameter a/b/c/d 이렇게 해줄래?

---

# Response

실행 완료. 좌측 하단이 3행 구조로 변경되었습니다:

```
         │ A │ B │ C │ D │
─────────┼───┼───┼───┼───┤
📈 Graph │ A │ B │ C │ D │  ← 드라이브별 오실로스코프 토글
🎮 Chart │ A │ B │ C │ D │  ← 드라이브별 컨트롤패널 토글
⚙ Param  │ A │ B │ C │ D │  ← 드라이브별 파라미터 패널 (기존)
```

- **Graph 행**: A/B/C/D 토글 시 해당 드라이브의 Oscilloscope 열기
- **Chart 행**: A/B/C/D 토글 시 해당 드라이브의 Control Panel 열기
- **Param 행**: 기존 A/B/C/D 파라미터 패널 토글 (IsPanelXVisible 바인딩 유지)
- 각 버튼은 드라이브별 PanelAccent 색상 적용 (A=빨강, B=파랑, C=초록, D=주황)
- 메시지에 `DriveId` 필드 추가하여 향후 멀티 인스턴스 확장 가능
