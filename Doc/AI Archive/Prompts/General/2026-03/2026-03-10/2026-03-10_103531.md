# Response

몇 가지 대안을 제안합니다:

**안 1) 드라이브 선택 + 기능 토글 분리**
```
┌──────────────────────────┐
│  A │ B │ C │ D           │  ← 드라이브 탭 (라디오 선택)
├──────────────────────────┤
│ 📈 Graph  │ 🎮 Chart    │  ← 선택된 드라이브에 적용
│ 📊 Param                 │
└──────────────────────────┘
```
A 선택 → Graph/Chart/Param 모두 A 드라이브 대상. 가장 깔끔하지만 동시에 A+B 그래프를 보려면 불편.

**안 2) 드라이브 탭 + 멀티 셀렉트 체크**
```
┌──────────────────────────┐
│ ☑A  ☑B  ☐C  ☐D          │  ← 멀티 선택 가능
├──────────────────────────┤
│ 📈Graph  🎮Chart  📊Param│  ← 체크된 드라이브 전체에 적용
└──────────────────────────┘
```
2행만 사용. A+B 체크 → Graph 누르면 A,B 둘 다 열림.

**안 3) 컴팩트 아이콘 매트릭스**
```
┌───┬──┬──┬──┬──┐
│   │A │B │C │D │
├───┼──┼──┼──┼──┤
│📈 │● │  │  │  │  ← 작은 원형 인디케이터
│🎮 │● │  │  │● │
│📊 │● │● │  │  │
└───┴──┴──┴──┴──┘
```
현재와 같은 매트릭스지만 셀 크기를 극도로 줄이고 아이콘만 사용.
