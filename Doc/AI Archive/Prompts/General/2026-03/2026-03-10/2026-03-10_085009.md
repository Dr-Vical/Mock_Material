# Prompt

컨트롤 패널이 눌려졌을 때 파란색으로 헤더가 칠해지는데 이거 바꿀 수 이썽?

---

# Response

실행 중입니다. 변경 사항 정리:

1. **AvalonDock 활성 탭 헤더** → 짙은 빨강(`#8B1A1A`) + 흰색 텍스트, 테마 전환 시에도 유지
2. **Enable 테두리 애니메이션** → 알파 60으로 훨씬 연하게 (이전 255 → 60)
3. **경고 다이얼로그 테두리** → `confirmBrushKey` 기반 자동 색상:
   - ZeroSet (PrimaryBrush) → 파란색 테두리
   - Clear Fault (WarningBrush) → 노란색 테두리
   - Reset (ErrorBrush) → 빨간색 테두리
4. **RS 로고** → 리본 상단 중앙에 SVG → XAML Path 변환 배치 (R/S 빨강 + Reliable&smart 회색 텍스트)
5. **앞으로 병렬 처리 가능한 작업은 병렬로 진행하겠습니다.**
