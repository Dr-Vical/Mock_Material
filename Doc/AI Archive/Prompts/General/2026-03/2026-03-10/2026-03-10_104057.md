# Prompt

2번으로 다시 적용해볼까?

---

# Response

실행 완료. 안 2 레이아웃으로 적용되었습니다:

```
┌──────────────────────────┐
│  [A]  [B]  [C]  [D]     │  ← 드라이브 멀티 선택 (체크 시 패널 색상 채움 + 하단 인디케이터 도트)
├──────────────────────────┤
│ 📈Graph │ 🎮Chart │ 📊Param │  ← 체크된 드라이브 전체에 적용되는 기능 토글
└──────────────────────────┘
```

**동작:**
- A+B 체크 → Graph 클릭 → A, B 둘 다 오실로스코프 열림
- A+B 체크 → Chart 클릭 → A, B 둘 다 컨트롤패널 열림
- A+B 체크 → Param 클릭 → A, B 파라미터 패널 토글
- 드라이브 선택 안 했으면 A가 기본 적용

확인해 보세요. 그래프/컨트롤패널 멀티 인스턴스 동시 배치는 현재 싱글 인스턴스 구조라 MainWindow 쪽 확장이 필요한데, 이 부분 진행할까요?
