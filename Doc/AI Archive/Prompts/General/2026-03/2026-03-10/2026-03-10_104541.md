# Response

실행 완료. 동작 방식:

```
평소:
┌─────────────────┐
│  A │ B │ C │ D  │  ← 하단 드라이브 버튼만 보임
└─────────────────┘

A 클릭:
┌─────────────────┐
│   Drive A       │  ← 스르륵 슬라이드 (200ms ease-out)
│ Graph│Chart│Param│  ← 3개 기능 버튼
├─────────────────┤
│ [A] │ B │ C │ D │  ← A가 패널 색상으로 하이라이트
└─────────────────┘

B 클릭: A 하이라이트 해제 → B 하이라이트 + "Drive B" 전환
A 다시 클릭: 패널 스르륵 접힘 (150ms ease-in)
```
