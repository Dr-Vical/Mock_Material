# Prompt

지금 폴던 안에 de es fr hu it ja 이건 언어팩 아냐? 이거 쓰고있어?

---

# Response

**안 쓰고 있습니다.** 그건 .NET 프레임워크/NuGet 패키지가 자동으로 뱉는 위성 어셈블리(satellite assembly)입니다.

| 폴더 | 내용 | 필요 여부 |
|------|------|----------|
| `ko/` | AvalonDock, WPF 프레임워크 한국어 리소스 | **필요** (한국어 UI) |
| `de/es/fr/hu/it/ja/...` 등 20여개 | 프레임워크 독일어/스페인어/불어 등 | **불필요** |
| `Localization/Strings.csv` | **우리 프로젝트** i18n (한→영) | **필요** |

프로젝트는 CSV 기반 자체 i18n 시스템(한국어/영어만)을 사용하므로, `ko/` 외 언어 폴더는 전부 용량 낭비입니다.

publish에서 불필요한 언어팩을 제거하려면 csproj에 한 줄 추가하면 됩니다. 적용할까요?

```xml
<SatelliteResourceLanguages>ko</SatelliteResourceLanguages>
```

이렇게 하면 `ko/`만 남고 나머지 언어 폴더가 전부 제거되어 publish 크기가 줄어듭니다.
