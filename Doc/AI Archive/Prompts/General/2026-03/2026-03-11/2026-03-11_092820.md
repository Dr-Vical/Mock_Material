DB 전환 질문에 답하면:

**지금은 CSV가 맞습니다.** 이유:

| 항목 | CSV (현재) | DB (SQLite 등) |
|------|-----------|----------------|
| 목적 | 목업/UI 설계 | 실제 제품 |
| 편집 | 메모장으로 즉시 수정 | 별도 툴 필요 |

**현재 CSV 데이터 구조:**

```
Config/ActionButtons.csv  → 트리노드별 액션 버튼 정의
Parameters/CSD7N_Parameter.csv → 드라이브 파라미터 목록
Localization/Strings.csv  → i18n 문자열
```

**DB 전환 시점** — 다음 조건이 생길 때:
- 파라미터 값의 **읽기/쓰기 이력 저장** 필요
- 다중 드라이브 설정의 **비교/버전 관리** 필요
- 사용자별 **커스텀 설정 저장** 필요
