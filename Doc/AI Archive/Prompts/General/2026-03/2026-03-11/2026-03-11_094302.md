# Prompt

응 필요해 파일 페이지 설정에는 작성 되어야 할 목록이 들어갈 거 같고, 인쇄 설정에는 위에 말한 것처럼 시스템 설정이 들어가야 할것 같아

# Response

실행했습니다. 리본 **파일** 탭에서 3개 버튼을 테스트해보세요:

- **페이지 설정** — 커스텀 다이얼로그 (문서 제목, 회사명, 드라이브 정보 포함 여부, 인쇄 항목 선택, 머리글/바닥글, 여백)
- **인쇄 설정** — Windows 시스템 프린터 설정 다이얼로그
- **인쇄** — 시스템 인쇄 다이얼로그 → 확인 시 FlowDocument로 파라메터 리포트 생성 후 인쇄

인쇄 리포트에 포함되는 내용:
- 문서 제목 + 회사명 + 날짜
- 드라이브 모델/펌웨어/연결 정보
- 파라메터 테이블 (전체 또는 변경분만, 패널별)
- 즐겨찾기 파라메터
- 변경된 값은 **굵은 빨간색**으로 강조
