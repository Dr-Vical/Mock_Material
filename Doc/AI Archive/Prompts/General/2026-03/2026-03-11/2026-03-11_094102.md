# Prompt

인쇄 페이지 설정 같은건 뭘 불러다 쓰는거야? 시스템에서? 아니면 생으로 만드는거야?

# Response

WPF에서는 두 가지입니다:

**시스템 제공 (바로 쓸 수 있는 것)**
- `System.Windows.Controls.PrintDialog` — Windows 기본 인쇄 다이얼로그 (프린터 선택, 매수, 범위 등)
- `PrintDialog.PrintTicket` — 용지 방향, 크기 등 기본 페이지 설정 포함

```csharp
var dlg = new PrintDialog();
if (dlg.ShowDialog() == true)
{
    dlg.PrintVisual(visual, "Parameter Export");  // UI 요소 그대로 인쇄
    dlg.PrintDocument(document, "Report");        // FlowDocument 인쇄
}
```

**직접 만들어야 하는 것**
- "페이지 설정" 전용 다이얼로그 (여백, 머리글/바닥글 등) — WPF에는 WinForms의 `PageSetupDialog` 같은 게 없음
- 인쇄 미리보기 — WPF 기본 제공 없음, `DocumentViewer` + `FixedDocument`로 직접 구성
- PDF 내보내기 — 별도 라이브러리 필요

**이 프로젝트에 넣으려면** 파라메터 목록 인쇄 정도라면 `PrintDialog` + `FlowDocument`로 충분합니다. 인쇄 미리보기나 PDF 변환까지 필요하세요?
