# Response

빌드 성공 (경고 3개는 기존 SkiaSharp 호환성 경고, 에러 0개).

완료된 변경 사항 정리:

### 1. 스케일 그룹 패널 (차트 좌측)
차트 좌측에 4개 스케일 그룹 추가:

| 그룹 | 채널 | 단위 | 색상 |
|------|------|------|------|
| **Current** | CH1 (Cont.Cur) + CH2 (Tgt.Cur) | A | ChartCH1Brush (노란) |
| **Velocity** | CH3 (Vel) + CH4 (Tgt.Vel) | rpm | ChartCH3Brush (파란) |
| **Position** | CH5 (Cur.Pos) + CH6 (Cur.Pos2) | pulse | ChartCH5Brush (주황) |
| **Status** | CH7 (Error) + CH8 (Inpos) | - | ChartCH7Brush (청록) |

각 그룹에:
- 색상 코딩된 좌측 보더 (해당 그룹 첫 채널 색상)
- **Max/Min TextBox** 편집 가능 (Enter 또는 포커스 이탈 시 적용)
- **Auto Scale 버튼** (그룹 데이터 범위 자동 계산)
- 채널 인디케이터 (색상 원 + 채널 이름)

### 2. 차트 테마 전환
- `RefreshChartTheme()` 메서드 추가 — ScottPlot 배경/글자/축/범례 색상을 현재 테마 토큰에서 재적용
- `SwitchTheme()`에서 자동 호출 → **화이트 테마에서 흰 배경 + 검정 글자로 전환됨**
