# Prompt

잘 되고 있고, 옆에 Read All 버튼 있잖아 거기에다가 Save Favorite / Load Favorite 이렇게 만들고 Save 할때는 창 하나 띄워서 프로그램 설치 폴더 하위에 Favorite 폴더 하나 만들어서 넣을 수 있게 파일 형태는 xaml이나 ini로

---

# Response

실행됐습니다. 구현된 기능:

1. **트리에서 "Favorites" 선택** → 액션 패널에 "Save Favorite" / "Load Favorite" 버튼 표시
2. **Save Favorite** → SaveFileDialog 열림, `{exe}/Favorite/` 폴더에 `.ini` 파일로 저장
3. **Load Favorite** → OpenFileDialog 열림, `.ini` 파일 로드 → FavoriteParameters 복원
4. **시작 시 자동 로드** → `Favorite/` 폴더에 `.ini` 파일이 있으면 자동으로 로드, 없으면 빈 상태

테스트: 별표 찍고 → Favorites 노드 클릭 → "Save Favorite" 버튼 → 파일 저장 → 앱 재시작 → 자동 로드 확인
