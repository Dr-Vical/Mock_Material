# Response

완료했습니다. 변경 요약:

### 1. 채널 선택 ComboBox
- 툴바 2행 구조: 상단(버튼), 하단(채널 선택기)
- 각 채널에 CheckBox + 색상 표시 + ComboBox (87개 파라메터 항목)
- 기본값: Motor Feedback Position / Master Position / Velocity Feedback / Velocity Command
- 선택 시 차트 범례 자동 갱신

### 2. Single / Continuous / Stop 버튼
- **Single**: 1회 측정 — 프로그래스 바 0→100% 채우며 데이터 수집 후 한 번에 표시
- **Continuous**: 연속 측정 — 기존 Start 동작 (실시간 스트리밍)
- **Stop**: 측정 중단

### 3. Option 버튼
- 클릭 시 `OscilloscopeOptionDialog` 팝업
- Sample Period / Samples per Division / Trigger Signal / Trigger Mode / Trigger Threshold / Pretrigger,%10 / Trigger Status — 모두 float 입력 필드

### 4. 프로그래스 바
- 툴바 우측 상단 배치
- Single 모드: 0→100% 채워지는 바
- Continuous 모드: Indeterminate 애니메이션
- "Collecting..." 텍스트 표시

### 5. "Favorite Parameter" 명칭
- 우측 패널 헤더: "Favorites" → "Favorite Parameter"
