# Response

실행 중입니다. 변경 내용:

1. **별표 동기화**: 패널 로딩 시 `FavoriteParameters`에 있는 항목은 자동으로 `IsFavorite = true` 설정 → DataGrid에 별표 표시
2. **드래그 순서변경**: 기존 코드 유지 (Favorites ListBox에서 텍스트 영역을 클릭 후 드래그하면 순서 변경 가능 — 별표 아이콘이 아닌 부분을 드래그)

확인해 주세요.
