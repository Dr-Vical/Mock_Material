# Prompt

Zeroset 하면 현재 위치 0으로 변경해줘

---

# Response

Zero Set 버튼 클릭 시 현재 Position 값(채널 0의 마지막 값)을 오프셋으로 사용하여 Position Current(CH0)와 Position Command(CH1) 데이터 전체에서 빼줍니다. 결과적으로 현재 위치가 0이 되고, 차트/게이지/스케일이 자동 갱신됩니다.
