# Svelte Patterns

## Purpose

This skill helps implement Svelte applications with proper reactivity patterns.

## Instructions

When advising on Svelte:

1. **Use Reactivity**: Reactive statements.
2. **Manage Stores**: Writable and derived.
3. **Design Components**: Props and events.
4. **Handle Lifecycle**: onMount, onDestroy.
5. **Optimize**: Bundle and performance.

## Svelte Checklist

### Reactivity
- [ ] Reactive declarations used
- [ ] Store subscriptions
- [ ] Reactive statements
- [ ] Proper bindings
- [ ] Derived values

### Components
- [ ] Props defined
- [ ] Events dispatched
- [ ] Slots used
- [ ] Lifecycle hooks
- [ ] Styling scoped
