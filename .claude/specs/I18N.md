# Internationalization (i18n) Specification — RswareDesign

> Last updated: 2026-02-24 | Migrated from Platform i18n (ko/en only)

## Summary

RswareDesign uses JSON-based localization with `DynamicResource` binding for runtime language switching. Keys follow `loc.{area}.{target}.{detail}` pattern. Two languages: Korean (ko) and English (en). Parameter names use CSV Name column (English) with Korean Remark where available.

---

## 1. System Architecture

```
Resources/Languages/ko.json    Resources/Languages/en.json
        ↓                                ↓
LocalizationService.Initialize()
        ↓
    ResourceDictionary (merged into App.Resources)
    "loc.menu.file" → "파일" / "File"
        ↓
    XAML: {DynamicResource loc.menu.file}  → updates at runtime
```

**Mechanism:** All translations stored in `ResourceDictionary` merged into `Application.Current.Resources`. XAML uses `{DynamicResource loc.*}` which auto-updates when `SetLanguage()` replaces dictionary entries.

---

## 2. Key Naming Convention

**Pattern:** `loc.{area}.{target}.{detail}`

| Prefix | Area | Example |
|--------|------|---------|
| `app.*` | Application-level | `loc.app.title`, `loc.app.version` |
| `menu.*` | Ribbon menu items | `loc.menu.file`, `loc.menu.file.save` |
| `menu.tools.*` | Tools tab | `loc.menu.tools.drive`, `loc.menu.tools.motor` |
| `menu.options.*` | Options tab | `loc.menu.options.font`, `loc.menu.options.theme` |
| `menu.connection.*` | Connection tab | `loc.menu.connection.rescan` |
| `menu.views.*` | Views tab | `loc.menu.views.driveTree`, `loc.menu.views.monitor` |
| `tree.*` | Drive tree | `loc.tree.online`, `loc.tree.offline`, `loc.tree.drive` |
| `tree.node.*` | Tree node names | `loc.tree.node.motor`, `loc.tree.node.pidTuning` |
| `param.*` | Parameter grid | `loc.param.header.ftnum`, `loc.param.header.value` |
| `action.*` | Action panel buttons | `loc.action.saveParam`, `loc.action.revert` |
| `monitor.*` | Monitor panel | `loc.monitor.title`, `loc.monitor.start` |
| `oscilloscope.*` | Oscilloscope | `loc.oscilloscope.title`, `loc.oscilloscope.trigger` |
| `error.*` | Error panel | `loc.error.title`, `loc.error.noErrors` |
| `status.*` | Status bar | `loc.status.connected`, `loc.status.mode.online` |
| `dialog.*` | Dialogs | `loc.dialog.serialSettings`, `loc.dialog.confirm` |
| `common.*` | Common buttons/labels | `loc.common.ok`, `loc.common.cancel`, `loc.common.save` |
| `theme.*` | Theme names | `loc.theme.dark`, `loc.theme.light` |
| `fault.*` | Fault messages | `loc.fault.overcurrent`, `loc.fault.overvoltage` |

**Rules:**
- All keys are lowercase with dot separators
- No hardcoded Korean/English in XAML — always use `{DynamicResource loc.*}`
- ko.json and en.json must have identical key sets (no orphan keys)

## 3. XAML Usage

```xml
<!-- Text content -->
<TextBlock Text="{DynamicResource loc.menu.file}" />

<!-- Button content -->
<Button Content="{DynamicResource loc.common.save}" />

<!-- Tooltip -->
<Button ToolTip="{DynamicResource loc.action.saveParam.hint}" />

<!-- Ribbon items -->
<fluent:RibbonTabItem Header="{DynamicResource loc.menu.file}" />
<fluent:Button Header="{DynamicResource loc.menu.file.save}" />

<!-- DataGrid column headers -->
<DataGridTextColumn Header="{DynamicResource loc.param.header.ftnum}" />

<!-- Window title -->
<Window Title="{DynamicResource loc.app.title}" />
```

## 4. ViewModel Usage

```csharp
public partial class MainWindowViewModel : ObservableObject
{
    private readonly ILocalizationService _loc;

    public string StatusText => _loc.Get("status.ready");

    public MainWindowViewModel(ILocalizationService loc)
    {
        _loc = loc;
        _loc.LanguageChanged += OnLanguageChanged;
    }

    private void OnLanguageChanged(object? sender, EventArgs e)
    {
        OnPropertyChanged(nameof(StatusText));
    }
}
```

## 5. Parameter Name Localization

Parameter names from CSV are primarily in English. Korean display uses Remark column where available.

| CSV Column | Usage |
|-----------|-------|
| `Name` | English parameter name (default display) |
| `Remark` | Korean description (used for ko locale tooltip or alternate display) |

`app.title`, `app.title.short`, `app.version`, `app.subtitle`

### Ribbon Menu (30+ keys)
- `menu.file` / `.open` / `.save` / `.load` / `.close`
- `menu.tools` / `.drive` / `.motor` / `.encoder`
- `menu.options` / `.font` / `.theme` / `.userMode`
- `menu.connection` / `.rescan` / `.serialSettings`
- `menu.views` / `.driveTree` / `.monitor` / `.oscilloscope` / `.errorLog` / `.controlPanel` / `.faults` / `.serviceInfo`

### Drive Tree (20+ keys)
- `tree.online`, `tree.offline`, `tree.drive`, `tree.group`
- `tree.node.*` (12+ node type names)
- `tree.group.basic` through `tree.group.auxiliary`

### Parameter Grid (10 keys)
- `param.header.ftnum`, `.name`, `.value`, `.unit`, `.default`, `.min`, `.max`
- `param.access.rw`, `param.access.r`
- `param.modified`

### Action Panel (8 keys)
- `action.saveParam`, `action.revert`, `action.setup`, `action.simpleDetail`, `action.close`, `action.help`
- `action.showHelps`, `action.showStatus`, `action.showCommands`

### Status Bar (8 keys)

### Common (10 keys)
